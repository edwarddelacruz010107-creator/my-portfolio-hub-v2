# Public Social Links Display Upgrade

- Displays every completed profile social link on the public portfolio.
- Supports GitHub, LinkedIn, Facebook, X/Twitter, Instagram, YouTube, Dribbble, and personal website.
- Empty fields are automatically hidden.
- Adds owner social links to project case-study pages.
- All external links open safely in a new tab with `noopener noreferrer`.
- Responsive two-column buttons collapse to one column on small screens.
