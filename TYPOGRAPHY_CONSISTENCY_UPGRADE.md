# Typography Consistency Upgrade

Applied a shared visual typography system to both tenant Admin and Superadmin dashboards while preserving the existing MyPortfolioHub identity.

## Font identity retained
- Display/headings: Syne
- UI/body/forms/tables: DM Sans
- Added real DM Sans 600 and 700 weights to prevent browser-generated synthetic bold.

## Standardized
- Page, section, card, and metric heading scales
- Sidebar labels and navigation weights
- Buttons and form controls
- Card radius and spacing rhythm
- Table headers, rows, and numeric alignment
- Badges, captions, helper text, and metadata
- Responsive typography on smaller screens

No backend logic, routes, database fields, or billing calculations were changed.
