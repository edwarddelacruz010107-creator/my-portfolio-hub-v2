Security Notes

Credential Storage

  • Keys & passwords encrypted at rest via Fernet
  • Never shown in plain text after saving
  • Gmail → use an App Password, not your main password

API Key Scope

  • Resend & MailerSend keys are send-only
  • OTP / password-reset use an isolated auth channel
  • Provider settings here do not affect auth email delivery
