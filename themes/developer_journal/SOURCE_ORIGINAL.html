About

Second-year IT student, building in public.

Hello! I'm John Edward Q. Dela Cruz, a second-year Information Technology student with more than one year of programming experience.

I began learning programming in April 2025 with Python. I later explored C and C++, then returned to Python to strengthen my understanding of programming fundamentals and problem-solving.

In March 2026, I started studying web development using HTML, CSS, and JavaScript. I am currently refreshing my C and C++ skills while learning Java and continuing to improve my Python and web development knowledge.

I enjoy building projects, solving problems, and discovering new technologies. My goal is to become a skilled software developer who creates practical, secure, and user-friendly applications.
